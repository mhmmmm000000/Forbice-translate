# 🌐 Translator Hub

A multi-translator web app featuring custom linguistic patterns and themes!

## 📖 Languages & Rules
- **Eis**: `what → ani les`, `sh → s`, `ch → ts`, `j/soft-g → dz`
- **Buwgeh**: `r → w`, `end-r → h` (Example: River → Wiveh)
- **Toothie**: Adds `th` to the end of every word (Example: Hello → Helloth)
- **Thighs**: Replaces trailing `s` with `th` for a lisp effect (Example: Yes → Yeth)
- **Nuss**: Cuts words in half and adds `nuss` (Example: Stuff → Stnuss)

## ✨ Features
- 🌗 **Themes**: Light, Dark, and pure AMOLED modes.
- 🔄 **Bi-directional**: Swap between English and your custom language instantly.
- 📱 **Responsive UI**: Clean tabbed interface.
